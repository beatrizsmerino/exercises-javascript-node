# APUNTES

## Commandos Git

— ——> Directorio en el q estas

```
ls
```

— ——> quit

```
q
```

— ——> entrar en el directorio

```
cd
```

— ——> Ir a la raiz

```
cd..
```

— ——> Inicializar repositorio git

```
git init
```

— ——> Añadir solo 1 archivo

```
git add fichero.txt
```

— ——> Añadir todo

```
git add .
git add -A
```

— ——> Ver estado

```
git status
```

— ——> Commit

```
git commit -m “Menssage”
```

— ——> Ver commits

```
git log
```

— ——> Mostrar y ocultar archivos
```
defaults write com.apple.finder AppleShowAllFiles TRUE KillAll Finder
```
